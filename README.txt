VETRIPREP — Deployment Package
================================

FILES:
  index.html       — Your full VETRIPREP app
  vercel.json      — Vercel deployment config
  FIREBASE_SETUP.html — Step-by-step Firebase setup guide
  README.txt       — This file

DEPLOY STEPS:
  1. Drag this entire folder to vercel.com
  2. Deploy → your site is live at vetriprep.vercel.app
  3. Follow FIREBASE_SETUP.html to enable sync

SYNC SETUP:
  1. Open FIREBASE_SETUP.html in browser
  2. Follow all 8 steps (takes ~10 min)
  3. Paste Firebase config into index.html
  4. Redeploy to Vercel
  5. Sign in with Google on any device — data syncs!
